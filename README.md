# 🌱 Green Campus Eco-System - Frontend

A modern web application developed to improve waste management within educational institutions. The system allows students to report waste issues, enables administrators to manage reports, and helps cleaning staff track and complete assigned tasks.

** Project Overview**

Green Campus Eco-System is a smart waste management platform designed to create a cleaner and more sustainable campus. The frontend provides an intuitive interface for students, administrators, and cleaning staff to interact with the system efficiently.

## ✨ Features

- Student Registration & Login
- Admin Login
- Cleaning Staff Login
- Report Waste with Image Upload
- Waste Report Tracking
- Dashboard for Administrators
- Reward Points System
- Campaign Management
- User Profile Management
- Responsive User Interface

## 🛠️ Tech Stack

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/aparna-apz003/WasteManageMent-FD.git
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```
